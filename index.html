C-Example Repository

Welcome to the C-Example repository by Bocaletto Luca.

About the Repository

This repository contains a collection of various C programming projects designed to illustrate fundamental and advanced concepts in C. Each project is developed as a one-page example with detailed inline comments to help you understand the language. All projects are fully documented and serve as practical exercises for learning and improving your skills in C programming.

Projects Included

  • Hello World Program: Basic program printing "Hello, World!" to the console.
  • Simple Calculator: Command-line calculator performing arithmetic operations.
  • Temperature Converter: Convert temperatures between Celsius and Fahrenheit.
  • Random Number Guessing Game: Guess the number game with hints.
  • Fibonacci Sequence Generator: Generate the Fibonacci sequence.
  • Prime Number Checker: Check if a number is prime using optimized loops.
  • File I/O: Read and Write: Demonstrates file operations in C.
  • Array-Based Stack Implementation: Stack data structure using an array.
  • Sorting Algorithms: Bubble Sort and Insertion Sort examples.
  • Searching Algorithms: Linear and Binary Search implementation.
  • Singly Linked List Implementation: Implement a singly linked list with CRUD operations.
  • Basic Banking System Simulation: Simulation of a banking system with persistent storage.
  • Tic Tac Toe Game: Text-based Tic Tac Toe game for two players.
  • Hangman Game: Classic Hangman game with word guessing logic.
  • Caesar Cipher for Encryption/Decryption: Encrypt and decrypt text using a Caesar cipher.
  • Student Record Management System: Manage student records with CRUD operations.
  • Memory Management and Pointer Exercises: Explore dynamic memory allocation and pointer arithmetic.
  • Multithreading with POSIX Threads: Demonstrate multithreading with mutex synchronization.
  • Basic Chatbot Program: Interactive chatbot using pattern matching.
  • Binary Search Tree (BST) Implementation: Implement a BST with various tree traversals.

Additional Projects

This repository also includes other extra projects that explore various additional topics and techniques in C programming.
